# Quantitative Finance Modeling with Python

This project implements quantitative finance models in Python for derivatives pricing, hedging, and portfolio risk analysis.

## Topics Covered
- Monte Carlo option pricing under Geometric Brownian Motion (GBM)
- Variance reduction techniques and convergence analysis
- Black-Scholes delta hedging simulations
- P&L analysis under discrete rebalancing
- CPPI and covered call strategy evaluation
- VaR and Expected Shortfall (ES) stress testing

## Tools
Python, NumPy, pandas, matplotlib

## Key Concepts
Derivative pricing, stochastic simulation, hedging error analysis, portfolio insurance, and tail-risk modeling.
